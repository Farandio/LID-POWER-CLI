## 🚀 Fitur Unggulan
* **AI Engine integration**: Akses langsung ke Architect 05 melalui perintah `ai [pesan]`.
* **Internal Navigation**: Navigasi `cd` yang stabil mendukung `..`, path panjang, dan `~`.
* **Powerful System Tools**:
    * `mk`: Membuat file (dengan ekstensi) atau folder baru secara instan.
    * `rm`: Penghapusan paksa (force delete) direktori atau file tanpa konfirmasi.
    * `kill-port`: Mematikan proses sistem pada port tertentu secara langsung.
* **Network & Security Suite**:
    * `ports`: Scanning port aktif yang sedang dalam status LISTEN.
    * `net`: Pengecekan status koneksi internet global.
    * `hash`: Pembuatan SHA256 integrity check untuk keamanan file.
* **Visual Hierarchy**: Visualisasi struktur folder menggunakan fitur `tree`.
